1. deploy: 部署配置，初始化脚本
2. pkg: 各个服务共享的代码包
3. service: 各个微服务的实现
    1. admin: 管理员后台服务
        1. cmd: 运行入口
        2. frontend: 前端代码
        3. config: 配置加载包
        4. configs: 配置文件存放处
        5. auth: 管理员登入功能核心组件，service，handler
        6. review: 预约审核功能实现的核心组件, service, handler, repository
    2. gateway: 微信信息推送以及身份鉴定
        1. api: rpc 服务 proto 消息的定义以及 rpc 实现代码
    3. reservation: 预约服务
    4. swagger: Swagger 文档聚合服务（端口 :8083）
4. skills: ai promot 封装
5. scripts: 构建脚本，辅助工具脚本
6. doc: 文档
7. docs: swagger 文档（swag init 自动生成）
8. tools: go 实现的辅助工具
