# 在本地开发测试
1. 在本地 docker 启动 redis, mysql 服务
```sh
# 启动 mysql, redis 的 Docker 服务
# 或者 bash script/env.sh up
docker-compose -f docker-compose.local.yaml up -d mysql redis
```

2. 使用脚本生成 token: openid 在 gen_token.go文件中设置
```sh
# openid 默认是 test_openid_local_001
# 或者 bash script/env.sh token
go run /cmd/tool/jwt/gen_token.go openid
```

3. 启动服务
```sh
# 或者 bash script/env.sh serve
go run cmd/api/v2/main.go --config configs/config_v2.debug.yaml

```

4. 运行请求测试脚本
```sh
bash script/test-api.sh
```

5. 清楚服务，mysql，redis
```sh
# 脚本只停止服务程序
bash script/env.sh down

# 卸载 mysql, reids 容器
docker-compose -f docker-compose.local.yaml down mysql redis
```
